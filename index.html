    Installation instruction

    Windows

    Just download lovalova.exe and launch it.

    Linux

    Open a terminal and write the following lines:

    sudo apt-get install love
wget http://lovasoa.github.io/lovalova/lovalova.love
love lovalova.love

    MacOS

    Download LÖVE for mac and the .love file. Then open the .love file with the LÖVE application.

    Downloads

    • Windows executable: lovalova.exe
    • .love file: lovalova.love

    Requirements

    • LÖVE (https://www.love2d.org)

    This game is written in lua, using the game framework LÖVE.
